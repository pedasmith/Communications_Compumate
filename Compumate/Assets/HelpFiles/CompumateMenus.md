# Compumate Menus

The complete Compumate menu system is below.

```
1 – Spelling Checker
2 – Calculator
3 – Telephone Directory
4 – Appointment Book
5 – Alarm Clock
6 – Personal File
7 – Typing Tutor
8 – PC Communications and Utilities
	1 – File Manipulation
		1 – Serial Send
            Items are WORD PROCESSOR, MONEY, PRIVATE FILE, APPOINTMENTS, TELEPHONE DIR, ALL. Items for which there is no data won't show up.
		2 – Serial Load
		3 – Read Cassette
		4 – Write Cassette
		5 – Verify Cassette
		6 – Memory Map (*example data*)
			WORD PROCESSOR : 141 bytes
			MONEY : 220 bytes
            TELEPHONE DIR : 9 bytes
            PRIVATE FILE : 177 bytes
            FREE RAM : 29278 bytes
		7 – Erase Program Data
			1 – WORD PROCESSOR
			2 – MONEY
			3 – TELEPHONE DIR.
			4 – PRIVATE FILE
	2 – Configuration
		1 – Serial Up/Download
			BAUD RATE: 50 ; 75 ; 110 ; 135 ; 150 ; 300 ; 600 ; 1200 ; 1800 ; 2400 ; 3600 ; 4800 ; 7200 ; 9600 ; 19200 
			DATA FORMAT : 8 bits, 1 stop ; 8 bits, 2 stop ; 7 bits, 1 stop ; 7 bits, 2 stop
            PARITY : none ; odd ; even ; mark ; space 
			HANDSHAKE:  none ; DTR/DSR ; XON/OFF ; DTR/DSR & XON/OFF
		2 – Serial Printer
			BAUD RATE: 50 ; 75 ; 110 ; 135 ; 150 ; 300 ; 600 ; 1200 ; 1800 ; 2400 ; 3600 ; 4800 ; 7200 ; 9600 ; 19200 
			DATA FORMAT : 8 bits, 1 stop ; 8 bits, 2 stop ; 7 bits, 1 stop ; 7 bits, 2 stop
            PARITY : none ; odd ; even ; mark ; space 
			HANDSHAKE:  none ; DTR/DSR ; XON/OFF ; DTR/DSR & XON/OFF
		3 – Serial Modem
			BAUD RATE: 50 ; 75 ; 110 ; 135 ; 150 ; 300 ; 600 ; 1200 ; 1800 ; 2400 ; 3600 ; 4800 ; 7200 ; 9600 ; 19200 
			DATA FORMAT : 8 bits, 1 stop ; 8 bits, 2 stop ; 7 bits, 1 stop ; 7 bits, 2 stop
            PARITY : none ; odd ; even ; mark ; space 
			HANDSHAKE:  none ; DTR/DSR ; XON/OFF ; DTR/DSR & XON/OFF
		4 – Parallel Printer
			DATA BITS: 8 data bits ; 7 data bits
		5 – Default Printer Port
			DEFAULT PRINTER: parallel printer ; serial printer
            PRINTER LINEFEED: LF after CR ; no LF after CR
		6 – Default Printer
			Imagewriter
            Brother HR-25
            Epson FX
            Epson MX
            Panasonic KXP
		7 – Dialing
			DIALING MODE: tone dialing ; pulse dialing
	3 – Help
9 – Word Processor
	C – Create Document
	E – Edit Document
	D – Delete Document
	P – Print Document
0 – Expense Account Module
	1 – New Entry
	2 – Review
	3 – Report
	4 – Set up
```
